Grapes Leaf Disease Deduction Web App

The Grapes Leaf Disease Deduction Web App is an innovative online tool designed to assist viticulturists, farmers, and plant health enthusiasts in identifying common grapevine leaf diseases such as Black Rot, ESCA, and Leaf Blight, as well as recognizing healthy leaves. By integrating a sophisticated machine learning model with an intuitive web interface.

Silkworm Disease Detection Web App

The Silkworm Disease Detection Web App is a state-of-the-art online tool developed to help sericulturists and researchers identify common silkworm diseases, including Flacherie, Grasserie, Muscardine, and Pebrine, as well as detect healthy silkworms. This web-based application employs a powerful machine learning model paired with a user-friendly interface to provide quick and precise disease identification.

Helmet Detection System

The Helmet Detection System is a Python-based application that utilizes a pre-trained YOLOv3 model and a custom Convolutional Neural Network (CNN) to detect motorbike riders without helmets in video footage. It processes a video file frame-by-frame, identifies motorbike riders, and determines whether they are wearing helmets.
